Fitur **Import Kontak** berfungsi untuk menambahkan data tamu secara otomatis dari daftar kontak pada perangkat smartphone.

- Fitur ini hanya dapat digunakan di perangkat **Android** maupun **iOS**.
- Khusus untuk perangkat **iPhone/iOS**, diperlukan konfigurasi tambahan agar fitur dapat berfungsi dengan baik.

### Pengguna Android

Untuk menggunkan fitur **Import Kontak** pada device Andriod diwajibkan untuk menggunakan browser **Chrome**.

### Pengguna iPhone/iOS

Ikuti langkah berikut untuk mengaktifkan akses kontak di iPhone/iOS:
